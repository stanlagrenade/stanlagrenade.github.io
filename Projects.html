Pacman in Motion
Project

This project demonstrates the power of JavaScript to generate animations by manipulating the DOM. The DOM is manipulated using loops, functions to show movement and screen edge detection is leveraged to create a bouncing effect. Finally, multiple images are leveraged and rotated to create the effect of movement with Pacman's mouth.

Eye Tracking
Project

This project demonstrates how the mouse cursor can be provide coordinates to trigger actions on screen like making the eyes follow the cursor the same way that your doctor makes you follow a pen during a checkup. This is achieved through manipulating the DOM.

MIT bus tracker
Project

This project demonstrates how APIs can be leveraged to integrate map data from Mapbox to create animations. This project calls functions triggered by a button to animate bus stops between MIT's Campus to Harvard's campus
